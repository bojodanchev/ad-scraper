# Platform Guide

## TikTok

### What You Can Scrape

TikTok scraping finds **viral organic videos** (not paid ads). Great for:
- Finding trending hooks and formats
- Discovering what's working in your niche
- Research creator content styles

### Search Types

| Type | How to Use | Example |
|------|-----------|---------|
| **Keyword Search** | Search TikTok like a user would | "AI automation", "dropshipping tips" |
| **Hashtag** | Search a specific hashtag | "ecommerce" (without #) |
| **Profile** | Scrape a creator's videos | "garyvee" (without @) |

### Filters Available

- **Time Period** - Last 48 hours, 7 days, 30 days, or 90 days
- **Min/Max Followers** - Filter by creator size (e.g., 10k-100k followers)
- **Sort By** - Latest, Oldest, or Most Popular

### Data You Get

For each video:
- Video URL (with and without watermark)
- Thumbnail image
- Caption and hashtags
- Likes, comments, shares, views
- Video duration
- Post date

For each creator:
- Username and display name
- Profile picture
- Bio
- Follower count
- Total likes received
- Verified status

## Instagram

### What You Can Scrape

Instagram scraping finds **organic posts and Reels** (not paid ads). Great for:
- Visual content inspiration
- Finding creators in your niche
- Tracking hashtag trends

### Search Types

| Type | How to Use | Example |
|------|-----------|---------|
| **Keyword/Hashtag** | Search posts by keyword | "fitness motivation" |
| **Hashtag Direct** | Scrape a specific hashtag | "entrepreneur" (without #) |
| **Profile** | Scrape a user's posts | "therock" (without @) |

### Filters Available

- **Time Period** - Filter by post date
- **Min Engagement Rate** - Only high-engagement posts (% of likes+comments/views)
- **Min Likes** - Filter by minimum likes
- **Min Views** - Filter by minimum views (for videos)

### Data You Get

For each post:
- Post URL
- Image/video URLs
- Caption
- Hashtags and mentions
- Likes, comments, views
- Post type (Image, Video, Carousel)
- Post date

For each creator:
- Username and display name
- Profile URL
- (Note: Follower counts require separate profile scrape)

## Meta Ads Library

### What You Can Scrape

Meta Ads Library shows **actual paid advertisements** running on Facebook and Instagram. This is official public data from Meta. Great for:
- Competitor research
- Ad copy inspiration
- Understanding what big brands are running

### Search Types

| Type | How to Use | Example |
|------|-----------|---------|
| **Keyword Search** | Find ads mentioning keywords | "AI automation", "weight loss" |
| **Page/Advertiser** | See all ads from a specific page | Page ID or URL |

### Filters Available

- **Country** - Target country (US, UK, DE, etc.)
- **Media Type** - All, Image only, Video only
- **Active Status** - Active ads only, Inactive, or All
- **Impressions** - Filter by reach (min/max)

### Data You Get

For each ad:
- Ad creative (images/videos)
- Headline and body copy
- Call-to-action button text
- Landing page URL
- Estimated reach (impressions range)
- Days running
- Target countries
- Start/end dates

For each advertiser:
- Page name
- Page URL
- Profile picture

## Platform Comparison

| Feature | TikTok | Instagram | Meta Ads |
|---------|--------|-----------|----------|
| Content Type | Organic videos | Organic posts | Paid ads |
| Engagement Data | Full (likes, views, etc.) | Full | Reach estimates only |
| Creator Stats | Detailed | Limited | Page name only |
| Best For | Viral hooks | Visual inspiration | Competitor ads |
| Scrape Time | 2-10 min | 2-10 min | 5-15 min |

## Tips for Each Platform

### TikTok Tips
- Use **keyword search** for niche research (e.g., "AI automation business")
- Filter by **popular** to see what's actually going viral
- Check creator follower counts - smaller creators (10k-100k) often have better engagement

### Instagram Tips
- Use **min engagement rate** filter to find high-performing content
- Scrape specific hashtags in your niche for targeted results
- Carousel posts often have the most saves/engagement

### Meta Ads Tips
- Search competitor brand names to see their current ads
- Filter to **active ads only** to see what's working now
- Check **days running** - ads running 30+ days are usually profitable
